# 🦞 Cowan – Die Buch-Instanz

**KI-Wissensassistent zum Buch "Agentic Authorship: Mit n8n, Claude und OpenClaw zur automatisierten Buchfabrik"**

> Cowan kennt das Buch in- und auswendig und beantwortet deine Fragen zu OpenClaw, Claude, n8n und mehr.

## 🚀 Live Demo

**[https://tikitackr.github.io/Cowan/](https://tikitackr.github.io/Cowan/)**

## ✨ Features

- **KI-Chat mit Quellenangabe** – Antworten basieren auf der Knowledge Base des Buches
- **BYOK (Bring Your Own Key)** – Nutze deinen eigenen Claude API-Key
- **Modellauswahl** – Haiku 4.5, Sonnet 4.5 oder Opus 4.5
- **Kostentracking** – Echtzeit-Anzeige von Token-Verbrauch und Kosten
- **Dark/Light Mode** – Automatisch oder manuell umschaltbar
- **Vorgeschlagene Fragen** – Einstiegsfragen zum schnellen Loslegen
- **Social Media Integration** – X/Twitter, WhatsApp, Telegram
- **Onboarding Tutorial** – Interaktiver Walkthrough fuer neue Nutzer
- **Markdown-Rendering** – Code-Bloecke, Listen, Links
- **100% clientseitig** – Laeuft komplett im Browser

## 🛠 Technologie

- **React 18** via CDN (kein Build-Tool noetig)
- **Claude API** (Anthropic) fuer die KI-Antworten
- **Standalone HTML** – Eine einzige Datei
- **GitHub Pages** fuer Hosting

## 📖 Ueber das Buch

"Agentic Authorship" zeigt, wie man mit n8n-Workflows, Claude als KI-Co-Autor und OpenClaw als Publishing-Plattform eine automatisierte Buchfabrik aufbaut. Cowan ist der interaktive Begleiter zum Buch.

## 🔑 Nutzung

1. Oeffne [https://tikitackr.github.io/Cowan/](https://tikitackr.github.io/Cowan/)
2. Gib deinen Claude API-Key ein
3. Stelle deine Frage zum Buch
4. Cowan antwortet mit Wissen aus der Knowledge Base

> Dein API-Key wird nur lokal im Browser verwendet und niemals gespeichert.
